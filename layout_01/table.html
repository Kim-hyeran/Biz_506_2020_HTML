Table tag를 이용한 데이터 리스트 표시

Table은 row, col을 지정하여 표 형식으로 데이터 리스트 등을 표현하는 Layout
Table에는 제목(칼럼)을 표시하는 thead가 있고, 내용(리스트, column)을 표시하는 tbody가 있다.
보통 간편하게 표를 만들 때는 thead와 tbody를 생략해도 된다.
**Table의 구조를 만들 때

  1. 먼저 몇 개짜리 row가 존재하는지 파악
  2. 지정된 row를 데이터가 표현하는 칼럼으로 나눈다
  3. row를 나누는 칼럼은 th, td tag를 사용하여 나눈다
  4. th, td 칼럼 사이에는 표시하고자 하는 데이터를 추가해준다
  5. th tag : table headline으로, 제목에 해당하는 굵은 글씨
  6. td tag : table data로 데이터들에 해당하는 일반 글씨
  7. 많은 데이터를 표시하고자 할 때는 tr, td.. tr 형식으로 계속해서 tr을 추가한다
  8. Table의 Border 속성 : Table의 테두리 선 지정
  9. width : 화면(Browser)에 대하여 Table 폭의 비율 조정
  10. width : %로 지정하면 화면 폭의 비율, px로 지정하면 고정 크기
  11. colspan : td의 개수가 다른 row와 다를 경우 td를 합하여 1개의 td로 병합
